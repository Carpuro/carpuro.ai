Cómo usarlo en tu sitio

Favicon (en tu <head>):

<link rel="icon" type="image/svg+xml" href="/assets/logos/carpuro-icon.svg"/>

Navbar (fondo oscuro):

<img
  src="/assets/logos/carpuro-logo-dark.svg"
  alt="carpuro.ai"
  width="200"
  height="50"
/>

Responsive con CSS (recomendado):

/* En tu CSS */
.logo {
  width: 180px;
  height: auto;
}

@media (max-width: 768px) {
  .logo { width: 140px; }
}

Inline SVG (máxima calidad, sin petición HTTP):

<!-- Copia el contenido de carpuro-logo-dark.svg directamente en tu HTML -->
<a href="/" aria-label="carpuro.ai home">
  <svg width="200" height="50" ... >...</svg>
</a>
